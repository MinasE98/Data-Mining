Α.¶

Για να βρούμε την παράμετρο της κατανομής που ταιριάζει στα δεδομένα των παρατηρήσεων χρησιμοποιώντας την Maximum Likelihood Estimation (MLE), πρέπει να κάνουμε τα ακόλουθα βήματα: 1.Βρίσκουμε τη συνάρτηση log-likelihood για τα δεδομένα n(λ) = logLn(λ) 2.Μέθοδος εφαπτομένης γραμμής: Βρίσκουμε την τιμή της παραμέτρου 𝜆 που μεγιστοποιεί τη συνάρτηση log-likelihood. Για να το κάνουμε αυτό, υπολογίζουμε την πρώτη και τη δεύτερη παράγωγο της συνάρτησης log-likelihood Τότε, υπολογίζουμε την νέα τιμή της παραμέτρου 𝜆 ως εξής: 𝜆_new = 𝜆_old - (1η παράγωγος) / (2η παράγωγος) 3.Επαναλαμβάνουμε το βήμα 2, χρησιμοποιώντας τη νέα τιμή της παραμέτρου 𝜆, μέχρι να φτάσουμε σε μία καλή εκτίμηση της παραμέτρου 𝜆. Μπορεί να είναι καλό να θέσουμε ένα όριο για το πόσο μικρή θα πρέπει να είναι η απόκλιση των νέων τιμών της 𝜆 από τις παλαιότερες, καθώς και ένα όριο για το πόσο πολλές επαναλήψεις θα κάνουμε τη μέθοδο της εφαπτομένης γραμμής. Όταν φτάσουμε σε μία καλή εκτίμηση της 𝜆, τότε θα έχουμε βρει την παράμετρο της κατανομής που ταιριάζει στα δεδομένα των παρατηρήσεων.Για να βρούμε την παράμετρο 𝜆 της εκθετικής κατανομής που ταιριάζει τα δεδομένα των παρατηρήσεων 𝑋 με τη μέθοδο Maximum Likelihood Estimation (MLE), ικανοποιήουμε την προδιαγραφή 𝑑𝐿(𝜆)/𝑑𝜆=0 ή 𝑑log(𝐿(𝜆))/𝑑𝜆=0. Συγκεκριμένα, μπορούμε να υπολογίσουμε τη συνάρτηση likelihood 𝐿(𝜆) ως το γινόμενο των πυκνοτήτων πιθανότητας 𝑓(𝑥) για όλες τις παρατηρήσεις 𝑋: 𝐿(𝜆)=∏𝑓(𝑥𝑖)=𝜆^𝑛 ∏𝑒^-𝜆𝑥𝑖 Μετατρέποντας τη συνάρτηση likelihood σε log-likelihood, έχουμε: log(𝐿(𝜆))=𝑛log(𝜆)+∑(-𝜆𝑥𝑖) Ικανοποιώντας την προδιαγραφή 𝑑log(𝐿(𝜆))/𝑑𝜆=0 και λύνοντας για 𝜆, βρίσκουμε: 𝜆_hat=∑𝑥𝑖/𝑛 Η καλύτερη εκτίμηση 𝜆_hat που ικανοποιεί την μέγιστη τιμή της συνάρτησης log-likelihood είναι λοιπόν 𝜆_hat=∑𝑥𝑖/𝑛. Σημείωση: Θα πρέπει να έχουμε υπόψη ότι η παράμετρος 𝜆 πρέπει να είναι μεγαλύτερη του μηδενός (𝜆>0), διαφορετικά η συνάρτηση likelihood δεν είναι έγκυρη.

B.¶

Για να βρούμε τις παραμέτρους 𝛩=(𝜆1,𝜆2,𝜋1,𝜋2) του mixture μοντέλου, πρέπει να κάνουμε τα ακόλουθα βήματα: 1.Αρχικά επιλέγουμε τυχαίες αρχικές τιμές για τις παραμέτρους 𝛩. 2.Για κάθε παρατηρήση 𝑥𝑖 και κάθε κατανομή 𝐿𝑘, υπολογίζουμε την πιθανότητα ανάθεσης 𝑃(𝐿𝑘|𝑥𝑖) ως εξής: 𝑃(𝐿1|𝑥𝑖)=𝜋1𝑓1(𝑥𝑖) / (𝜋1𝑓1(𝑥𝑖)+𝜋2𝑓2(𝑥𝑖)) 𝑃(𝐿2|𝑥𝑖)=𝜋2𝑓2(𝑥𝑖) / (𝜋1𝑓1(𝑥𝑖)+𝜋2𝑓2(𝑥𝑖)) 3.Υπολογίζουμε τις νέες τιμές 𝜆1,𝜆2,𝜋1,𝜋2 ως εξής: 𝜆1=Σ(𝑃(𝐿1|𝑥𝑖)*𝑥𝑖) / Σ(𝑃(𝐿1|𝑥𝑖)) 𝜆2=Σ(𝑃(𝐿2|𝑥𝑖)*𝑥𝑖) / Σ(𝑃(𝐿2|𝑥𝑖)) 𝜋1=Σ(𝑃(𝐿1|𝑥𝑖)) / 𝑛 𝜋2=Σ(𝑃(𝐿2|𝑥𝑖)) / 𝑛 4.Επαναλαμβάνουμε τα βήματα 2 και 3, μέχρι να φτάσουμε σε μία καλή εκτίμηση των παραμέτρων 𝛩. Μπορεί να είναι καλό να θέσουμε ένα όριο για το πόσο μικρή θα πρέπει να είναι η απόκλιση των νέων τιμών των 𝛩 από τις παλαιότερες, καθώς καθώς και ένα όριο για το πόσο πολλές επαναλήψεις θα κάνουμε τον ΕΜ αλγόριθμο. Όταν φτάσουμε σε μία καλή εκτίμηση των παραμέτρων 𝛩, τότε έχουμε βρει το καλύτερο mixture μοντέλο που ταιριάζει στα δεδομένα 𝑋.

df_train = pd.read_csv('phily_users_businesses_stars_train.csv')
user_averages = df_train.groupby('user_id')['stars'].mean()
business_averages = df_train.groupby('business_id')['stars'].mean()

def predict_rating_ua(user, business):
    user_average = user_averages[user]
    return user_average

def predict_rating_ba(user, business):
    business_average = business_averages[business]
    return business_average

from sklearn.decomposition import TruncatedSVD
K = 100
df_train = pd.read_csv('phily_users_businesses_stars_train.csv')
df_test = pd.read_csv('phily_users_businesses_stars_test.csv')
user_ids = df_train['user_id'].unique()
business_ids = df_train['business_id'].unique()
user_id_map = {id: index for index, id in enumerate(user_ids)}
business_id_map = {id: index for index, id in enumerate(business_ids)}
R = np.zeros((len(user_ids), len(business_ids)))

for row in df_train.itertuples(index=True, name='Pandas'):
    user_id = getattr(row, "user_id")
    business_id = getattr(row, "business_id")
    rating = getattr(row, "stars")
    R[user_id_map[user_id], business_id_map[business_id]] = rating

rmse_values = []
time_values = []

for k in range(1, K+1):
    start_time = time.time()
    svd = TruncatedSVD(n_components=k)
    R_k = svd.fit_transform(R)
    R_k = np.dot(R_k, svd.components_)
    predictions = []
    for row in df_test.itertuples(index=True, name='Pandas'):
        user_id = getattr(row, "user_id")
        business_id = getattr(row, "business_id")
        rating = getattr(row, "stars")
        predicted_rating = R_k[user_id_map[user_id], business_id_map[business_id]]
        predictions.append(predicted_rating)

    rmse = np.sqrt(mean_squared_error(df_test['stars'], predictions))
    rmse_values.append(rmse)
    time_values.append(time.time() - start_time)
    
plt.plot(range(1, K+1), rmse_values)
plt.xlabel('k')
plt.ylabel('RMSE')
plt.show()

plt.plot(range(1, K+1), time_values)
plt.xlabel('k')
plt.ylabel('Time (s)')
plt.show()

import numpy as np

def item_based_cf(R, k):
    pred = np.zeros_like(R)  
    for u in range(R.shape[0]):
        for i in range(R.shape[1]):
            if R[u, i] == 0:  
                if len(R[u].nonzero()[0]) == 0:
                    continue
                sim = cosine_similarity(R[:, i], R[:, R[u].nonzero()[0]])  
                k_most_sim = np.argsort(sim)[-k:]  
                pred[u, i] = np.mean(R[u, k_most_sim])  
    return pred

def cosine_similarity(a, b):
    sim = np.dot(a, b.T) / (np.linalg.norm(a) * np.linalg.norm(b, axis=1))
    return sim

k_values = [1, 2, 3, 5, 8, 10, 12, 15, 20, 25, 30, 40, 50]
rmse = []
test_mask = np.zeros_like(R)
test_mask[df_test['user_id'].astype(int)-1, df_test['business_id'].astype(int)-1] = 1

for k in k_values:
    pred = item_based_cf(R, k)
    rmse.append(np.sqrt(np.mean((pred[test_mask == 1] - R[test_mask == 1])**2)))


import matplotlib.pyplot as plt
plt.plot(k_values, rmse)
plt.xlabel('k')
plt.ylabel('RMSE')
